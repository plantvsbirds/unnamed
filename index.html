Unnamed

this project is not unnamed. it's called 'unnamed'. Unnamed is a modern web solution for tracking dark traffic.

Any web page served with Unnamed middleware will include certain user info. When a user copy certain paragraph of text and distribute it on other channel, user info can be retrived from the copied paragraph(as long as it's long enough), ideally only by admin of the site.

This is only a small demo for Unnamed. Unnamed could also track image & screenshot using FFT. Here we have a text version done w/ a simple scheme & Unicode confusion table crafted by the team(Hong Lin).
